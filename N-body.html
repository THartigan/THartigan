Overview

  • Researched and implemented the fast multipole and Barnes‑Hut algorithms for N‑body simulation.
  • Compared the computational complexity of these algorithms and found both to scale as 𝑂(𝑁 log 𝑁 ), beating direct calculations of 𝑂(𝑁^2).
  • Found that the Barnes‑Hut algorithm achieved much better accuracy in a given time despite both algorithms being of the same computational complexity.
  • For more details, see my Project Report or view the code on GitHub.
