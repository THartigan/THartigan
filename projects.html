Identification of Situp Motion Using Neural Networks

Ongoing

An ongoing project. Milestones reached so far:

  • Developed apps in swift for Apple Watch and iPhone. This allows gyroscope data recorded by the watch to be synchronised with a video recorded by the iPhone. The iPhone app then allows for playback of the video to label important events.
  • Created a machine learning framework in python using PyTorch which takes a dataset recorded using these apps and trains a model to classify whether a sit-up was completed in a short timeframe or not. This also rejects "fake" sit-ups.
  • Currently developing an app which uses this model to count the number of sit-ups completed locally on an Apple Watch. This aspect currently works, and will be expanded into a useful app soon.
